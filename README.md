# 🎮 Super Trunfo em C

Projeto desenvolvido em linguagem C com foco no aprimoramento da lógica de programação, manipulação de dados e comparação de atributos.

## 🚀 Descrição

Este projeto simula um jogo de Super Trunfo, onde duas cartas são cadastradas com diferentes atributos e comparadas entre si com base na escolha dos jogadores.

O sistema permite selecionar atributos específicos para disputa e determina o vencedor com base nos valores informados.

---

## 🧠 Funcionalidades

- Cadastro de duas cartas com dados personalizados
- Cálculo automático de:
  - Densidade populacional
  - PIB per capita
  - Super Poder (pontuação geral)
- Escolha de atributos pelos jogadores
- Comparação dinâmica entre cartas
- Exibição do vencedor com base nas regras do jogo

---

## 🛠️ Tecnologias utilizadas

- Linguagem C
- Biblioteca padrão (`stdio.h`, `string.h`)

---

## 📚 Conceitos aplicados

- Estruturas condicionais (`if`, `switch`)
- Operadores ternários
- Entrada e saída de dados
- Manipulação de strings
- Cálculos matemáticos
- Lógica de comparação
