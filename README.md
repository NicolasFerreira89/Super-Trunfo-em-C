# 🎮 Super Trunfo em C

Projeto desenvolvido em linguagem C com o objetivo de praticar lógica de programação e conceitos fundamentais.

## 🚀 Funcionalidades
- Comparação de atributos entre cartas
- Entrada e saída de dados
- Lógica condicional para determinar o vencedor

## 🧠 Conceitos aplicados
- Estruturas 
- Condicionais (< e >)
- Manipulação de dados
- Organização de código

## ▶️ Como executar
1. Compile o código:
   gcc main.c -o jogo

2. Execute:
   ./jogo
